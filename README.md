A Python-based virtual assistant that listens for the wake word **"Jarvis"** and executes voice commands such as opening websites and playing music.

## Features
- Voice recognition with `speech_recognition`
- Text-to-speech responses using `pyttsx3`
- Hands-free automation of tasks like:
  - Open Google, YouTube, LinkedIn, Facebook
  - Play songs from a custom `musiclibrary.py`
- Wake-word activation ("Jarvis") for accessibility
